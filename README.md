# ApplicationNotes
```СТРУКТУРА РАБОТЫ ПРОГРАММЫ:```
1. Запуск осуществляется из файла **clientNotepad.py**
2. Алгоритм выполнения:
  1. Первый раз пользователю выводится информация для действий;
  2. Запускает бесконечный цикл, выход из которого предусмотрен командой --->> exit;
  3. Предлагаются варианты работы с блокнотом заметок и необходимыми командами;
  4. В случае неверного ввода данных, осуществляется выход в главное меню;
  5. Операции удаления (редактирования) данных осуществляются только после поиска данных;
  6. Удалять можно несколько заметок, как и редактировать;
  7. В случае редактирования нескольких заметок, изменения заголовка (сообщения), в том числе времени изменения, будет производиться во всех редактируемых заметках;
  8. В случае, когда пользователю предлагается ввести число для выполнения операции, или нажать кнопку Enter, то любые данные, введенные пользователем, отличные от цифры (команды), будут интерпретированы программой, как Enter

## ОПИСАНИЕ РАБОТЫ ФУНКЦИЙ:
1. ***searchNotes()***:
  1. принимает список от getListFromJson();
  2. принимает от пользователя данные (искомую дату);
  3. фильтрует список по данным, введенным пользователем;
  4. если список пуст, предлагает:
                                  -осуществить новый поиск (повторный вызов функции);
                                  -вернуться в главное меню (завершить работу);
  5. если список содержит данные, предлагает:
                                  -осуществить новый поиск (повторный вызов функции);
                                  -продолжить действия с искомыми данными (возвращает искомый список (заметку(заметки));

2. ***getNotepad()***:
  1. принимает список от getListFromJson()
  2. если список пуст, тысячекратно просит прощения, что нет данных
  3. если список содержит данные:
                                - сортирует функцией sortedData(data)
                                - выводит в консоль функцией printNotepad(data)

3. ***addNote()***:
  1. циклом принимает от пользователя ввод не пустого заголовка;
  2. циклом принимает от пользователя ввод не пустого сообщения;
  3. функцией getId() принимает новый id;
  4. функцией datetime.datetime.today() принимает время создания заметки
  5. создает новую заметку с данными пунктов №№ 1-4
  6. осуществляет дозапись в json-файл новой заметки

4. ***deleteNotes(useDataDelete)***:
  1. принимает на вход список, предназначенный для удаления;
  2. создает множество id, которые содержит список для удаления
  3. принимает список (блокнот заметок) от getListFromJson();
  4. создает копию блокнота заметок для перезаписи в json-файл;
  5. циклом проходит по оригинальному блокноту (если находит совпадение ключа во множестве), то удаляет текущий элемент копии блокнота
  6. функцией outInJsonData(dataForWrite) перезаписывает данные из отредактированной копии блокнота

5. ***editNotes(useData)***:
  1. на вход принимает список для редактирования;
  2. функцией useCommand(nameField) принимает команду действия от пользователя (редактировать или нет);
  3. если редактировать, то с помощью функции notEmpty(nameField) получает данные от пользователя и записывает эти данные в поле "title" каждого элемента списка;
  4. демонстрирует пользователю результат (текущее состояние редактируемых данных));
  5. проводит операции пунктов №№ 2-4 для поля "msg" каждого элемента списка;
  6. если изменение одного из полей ("title" или "msg") было произведено:
      1. в поле "date" каждого элемента списка перезаписывается текущее время;
      2. принимает список (блокнот заметок) от getListFromJson();
      3. создает множество id, которые содержит список отредактированных данных
      4. циклом проходит по блокноту (если находит совпадение ключа во множестве), то присваивает текущему элементу элемент из списка отредактированных данных
      5. функцией outInJsonData(notepad) сохраняет измененный блокнот в json-файл;
      6. демонстрирует пользователю отредактированные данные, которые сохранены в json-файл;
  7. иначе сообщает пользователю, что никаких изменений нет
  8. содержит функции:
      *notEmpty(nameField)*:
        1. принимает на вход имя (заголовок, сообщение)
        2. проверяет вводимые пользователем данные на пустую строку
        3. если строка пустая - повторно вызывает себя
        4. возвращает не пустую строку
      *useCommand(nameField)*:
        1. принимает на вход имя (заголовок, сообщение)
        2. просит от пользователя ввода команды
        3. возвращает команду

6. ***printNotepad(list)***:
  1. принимает на вход список;
  2. выводит в консоль согласно установленному образцу (разделение полей через ;);

7. ***getId()***:
  1. принимает список (блокнот заметок) от getListFromJson();
  2. если список не пустой, то ищет максимальный id, добавляет единицу;
  3. возвращает новый (максимальный) уникальный индекс по текущему состоянию списка;

8. ***getListFromJson()***:
  1. считывает данные из json-файла в список;
  2. возвращает список;

9. ***outInJsonData(useDate)***:
  1. принимает на вход список, предназначенный для записи
  2. перезаписывает данные в json-файл

10. ***sortedData(list)***:
  1. принимает на вход список;
  2. сортирует список по дате (в порядке от ранней даты к текущему времени);
